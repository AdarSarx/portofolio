Portfolio statis modern untuk mahasiswa Teknologi Informasi dengan section lengkap: Hero, About, Skills, Projects, PKL Experience, Organization, Certificates, dan Contact. Termasuk dark/light toggle dan layout responsif.

## Quick Start

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Preview Build

```bash
npm run preview
```

## Deploy to Vercel

1. Push repository ke GitHub.
2. Import project di Vercel.
3. Deploy otomatis dari branch utama.

## Customize Content

Edit konten utama di `src/App.jsx` dan styling di `src/App.css`.
